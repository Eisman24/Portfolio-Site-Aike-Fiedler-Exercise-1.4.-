My Process: User Research

As part of the UX Design course with Career Foundry I created a web app for Stickerstars to enrich the digital experience and make the process from applying to creating your own scrapbook more consistent for our users. The tasks included user research, wireframing, prototyping, user testing and visual design. We will have a closer look at User Research in this part.

User Personas

The goal here is to create a believable, living character, which helps to solve real problems and create meaningful services and products for relevant users. In order to create a memorable and more human persona, we add extra details that help to tell the stories of its represented users. 

User Journey

User journey maps are a great tool for visualizing processes a user goes through in order to accomplish a goal. They also can be very effective in a group environment. By being involved in the process, every stakeholder can learn a lot about your user and product.

User Flow

User flows are an amazing tool for discovering what pages or screens of our app are needed and which not.User flows help us designers avoid a common mistake in product development, so that we design more user centered and not just self centered. Alongside we have one user flow from our Stickerstars Application.

Conclusion

Its been a great expierence diving deeper into the topic of user research. I got a better understanding valuable it can be and also how easy it could be done.
